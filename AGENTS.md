## Project Goal

Build a premium Next.js + Vercel website for Francis Petcare Hospital that improves local SEO, trust, calls, and appointment bookings.

## Critical Rules

- No bird care unless confirmed.
- No 24/7 emergency claim.
- No fake reviews.
- No invented doctors, reviews, credentials, awards, ratings, or medical claims.
- Use verified doctor/review data only.
- Use Francis logo colors.
- Primary CTA must be deep navy/medical blue.
- Avoid random green CTA buttons.
- Avoid generic veterinary templates.
- Avoid SaaS/pet-shop/coupon style.
- Run npm run build after implementation changes.

## Workflow

For major changes:
1. Read docs.
2. Summarize understanding.
3. Plan.
4. Wait for approval if requested.
5. Implement small scope only.
6. Run build.
7. Summarize files changed and QA notes.
