
# Francis Petcare AI Project Memory

Before coding, read this file and all referenced docs.

## Project Goal

Build a premium Next.js + Vercel website for Francis Petcare Hospital in Richmond, BC.

The site must improve:
- local SEO rankings
- trust within 5 seconds
- calls
- appointment bookings
- exotic pet authority
- Richmond location relevance
- mobile conversion

## Platform Direction

We are rebuilding in Next.js + Vercel instead of continuing to patch the messy WordPress site.

The current WordPress site remains live until the new site is approved and launched.

## Critical Rules

- Do not mention bird care unless confirmed.
- Do not claim 24/7 emergency care.
- Use “urgent care during clinic hours,” “same-day appointments,” or “call ahead.”
- Do not invent doctors, reviews, credentials, awards, star ratings, medical claims, or testimonials.
- Do not use fake reviews.
- Do not use review gating language.
- Do not show old Catcare-era reviews as primary proof.
- Do not create a generic veterinary template.
- Do not make the site look like a SaaS landing page, pet shop, coupon site, or cheap AI template.
- Use the Francis Petcare logo as the brand anchor.
- Use deep navy/medical blue as the primary CTA color.
- Do not use random green CTA buttons.
